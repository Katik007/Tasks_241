# Управление пользователями

1) Добавьте пользователей user1 и user2:
    1.1) user1 - оболочка bash
    sudo useradd -m -s /bin/bash user1
    1.2) user2 - оболочка sh
    sudo useradd -m -s /bin/sh user2
    1.3) установите им пароли
    sudo passwd user1
    sudo passwd user2
